# TakeUForward Round 2 — Dashboard UI

A responsive React dashboard I built as part of the TakeUForward Round 2 assignment.

## What it is

It's a dashboard layout with four sections — a top banner with company stats, a milestone/activity card, an AI enrichment card, and a CRM integration card. The design is based on a reference image and built entirely with React and plain CSS.

## Tech used

- React 18
- Vite
- Plain CSS (no UI library)

## Run it locally

```bash
npm install
npm run dev
```
